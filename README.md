# Casino Nights — dealmyevent.com

GitHub Pages site for Casino Nights event entertainment.

## Deploy
1. Upload all files to repo root
2. Settings → Pages → Deploy from branch: main
3. Custom domain: dealmyevent.com

## Tally Form
Form embedded from: https://tally.so/r/zxzbWa
To change the form, update the `data-tally-src` iframe URL in index.html.
