# DealMyEvent — Casino Event Landing Page

This repository contains the source for the DealMyEvent landing page, a lightweight promotional website for booking casino-style entertainment at private events.

The site is built as a simple static webpage and hosted with GitHub Pages. It is designed to be fast, easy to edit, and inexpensive to host while still providing a professional booking experience.

## Overview

The goal of this site is straightforward: provide a clear introduction to the casino party services offered and give visitors a simple way to request a quote for their event.

The page highlights:

* Available casino games
* Example event packages
* Testimonials
* A quote request form

All booking inquiries are handled through an embedded Tally form which sends submissions directly by email.

## Tech Stack

This project intentionally avoids heavy frameworks to keep things simple and reliable.

* HTML5
* CSS3
* Vanilla JavaScript
* GitHub Pages for hosting
* Tally.so for form submissions

Because everything is static, the site loads quickly and has minimal maintenance requirements.

## File Structure

```
index.html
style.css
script.js
favicon.ico
README.md
```

`index.html`
Main website layout and content.

`style.css`
Handles styling and layout.

`script.js`
Small scripts used for page behavior.

## Editing the Site

Most updates can be made by editing `index.html` and committing the changes.

Typical edits include:

* Updating package descriptions
* Adjusting testimonials
* Updating contact or booking information
* Changing text or images

Once changes are committed and pushed to the main branch, GitHub Pages automatically deploys the updated site.

## Contact Form

The inquiry form is powered by Tally.

Form submissions are delivered via email so event requests can be followed up on quickly.

## Hosting

The site is hosted through GitHub Pages and connected to the custom domain:

```
dealmyevent.com
```

DNS is managed through Porkbun.
